# A RESTful Demo

A demo for how to use the RESTful programming style.

## What it Does

* Covers `GET`, `POST`, `PUT`, `DELETE`.
    * `GET` : retrieves data.
    * `POST` : creates data.
    * 'PUT' : updates data.
    * 'DELETE': removes data.
* All of the functionality is provided through `.htaccess` rules.
